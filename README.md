This was a home defence mini application developed to run on IoT devices, notably the Raspberry Pi. Users can take photos using a USB camera connected to the Raspberry Pi, which users can install around there house. 
Users can connect via a web interface and take photos of the camera. These will then be stored in the users dropbox account.

The project serves as a proto-type and proof of concept. Majour enhancements and will potentially be implemented in the future.
